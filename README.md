# N-Queens Problem Solver

A clean and efficient Python implementation to solve the classic **N-Queens Puzzle** using backtracking. 

---

## 📌 Problem Description
The **N-Queens puzzle** is the problem of placing $N$ chess queens on an $N \times N$ chessboard so that no two queens threaten each other. This means:
* No two queens can be in the same **row**.
* No two queens can be in the same **column**.
* No two queens can be on the same **diagonal**.

---

## 🚀 Features
* **Backtracking Algorithm:** Uses an optimized recursive approach to find valid configurations.
* **Dynamic Board Size:** Works for standard $8\times8$ or any customizable $N \times N$ board.
* **Visual Output:** Prints the chessboard in a clear text-based format showing the exact positions of the queens.
